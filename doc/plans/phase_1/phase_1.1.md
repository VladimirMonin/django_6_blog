# Фаза 1.1: Инициализация Django 6 проекта

## Цель
Создать базовый Django 6 (beta) проект с одним приложением для блога.

## Задачи

### 1. Создание Django проекта
- [ ] Инициализировать Django 6 проект через Poetry
- [ ] Настроить `pyproject.toml` с зависимостями
- [ ] Создать конфигурационный пакет проекта (настройки, urls)
- [ ] Проверить работоспособность сервера

### 2. Создание приложения блога
- [ ] Создать Django-приложение `blog`
- [ ] Подключить приложение в `INSTALLED_APPS`
- [ ] Создать базовую структуру папок в приложении

### 3. Базовая конфигурация
- [ ] Настроить `TEMPLATES` для работы с шаблонами в корне проекта
- [ ] Настроить `STATIC_URL` и `STATICFILES_DIRS`
- [ ] Создать папку `templates/` в корне проекта
- [ ] Создать папку `static/` в корне проекта

## Результат
Рабочий Django 6 проект с приложением `blog`, готовый к разработке функционала.

## Зависимости
- Python 3.11+
- Poetry 2
- Django 6.0 (beta)

# Фаза 1: Базовое Django 6 приложение с Bootstrap 5

## Общая цель

Создать базовое Django 6 (beta) приложение блога с функционалом просмотра списка и детального просмотра постов, интегрированное с Bootstrap 5.

## Результат фазы

Работающее SSR-приложение на Django 6 с:

- Моделью Post и тестовыми данными
- Представлениями списка и детального просмотра
- Шаблонами в корне проекта
- Bootstrap 5 стилизацией по style_guide
- Django 6 Template Partials для модульности
- django-components для переиспользуемых UI-компонентов
- HTMX для динамического взаимодействия без перезагрузки страниц

## Технологический стек

### Базовые технологии
- Django 6.0 (beta) — серверная часть
- Bootstrap 5.3 — CSS фреймворк
- Python 3.11+ — язык программирования
- Poetry 2 — управление зависимостями

### Современные инструменты
- **Django 6 Template Partials** — модульные шаблоны без лишних файлов
- **django-components** — переиспользуемые UI-компоненты с инкапсуляцией
- **HTMX** — динамические взаимодействия через HTML-атрибуты
- **django-htmx** — интеграция HTMX с Django

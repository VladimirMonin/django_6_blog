# Фаза 1: Базовое Django 6 приложение с Bootstrap 5

## Общая цель
Создать базовое Django 6 (beta) приложение блога с функционалом просмотра списка и детального просмотра постов, интегрированное с Bootstrap 5.

## Результат фазы
Работающее SSR-приложение на Django 6 с:
- Моделью Post и тестовыми данными
- Представлениями списка и детального просмотра
- Шаблонами в корне проекта
- Bootstrap 5 стилизацией по style_guide
- Готовой структурой для интеграции HTMX (фаза 2)

## Подфазы

### [Фаза 1.1](phase_1.1.md): Инициализация Django 6 проекта
Создание базового проекта, приложения `blog`, настройка структуры папок.

### [Фаза 1.2](phase_1.2.md): Настройка маршрутизации
Создание URL-паттернов для списка постов, детального просмотра, страницы "О блоге".

### [Фаза 1.3](phase_1.3.md): Создание моделей и датасета
Модель Post, миграции, management команда для генерации 10-15 тестовых постов.

### [Фаза 1.4](phase_1.4.md): Создание представлений и шаблонов
Реализация view-функций и шаблонов для списка и детального просмотра.

### [Фаза 1.5](phase_1.5.md): Интеграция Bootstrap 5
Подключение Bootstrap 5, стилизация по концепции (белый фон, черные кнопки, желтые акценты).

## Технологический стек
- Django 6.0 (beta)
- Bootstrap 5.3
- Python 3.11+
- Poetry 2

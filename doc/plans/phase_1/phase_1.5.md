# Фаза 1.5: Интеграция Bootstrap 5 и базовая стилизация

## Цель
Подключить Bootstrap 5 и применить базовую стилизацию согласно style_guide.

## Задачи

### 1. Подключение Bootstrap 5
- [ ] Добавить в `base.html`:
  - CDN-ссылку на Bootstrap 5.3 CSS
  - CDN-ссылку на Bootstrap 5.3 JS Bundle
  - Настроить viewport meta-тег

### 2. Базовая структура layout
- [ ] Обновить `base.html`:
  - Добавить контейнер Bootstrap (container или container-fluid)
  - Создать блок навигации с Bootstrap Navbar
  - Применить белый фон (bg-white)
  - Добавить основной контент в Bootstrap grid

### 3. Стилизация навигации
- [ ] Создать navbar с:
  - Логотипом/названием блога
  - Ссылками: Главная, О блоге
  - Использовать желтый акцент (bg-warning) для активного пункта
  - Темные кнопки (btn-dark) при необходимости

### 4. Стилизация списка постов
- [ ] В `post_list.html`:
  - Использовать Bootstrap cards для каждого поста
  - Применить grid систему (row/col)
  - Кнопки "Читать далее" — btn btn-dark
  - Даты и метаданные — text-muted

### 5. Стилизация детального просмотра
- [ ] В `post_detail.html`:
  - Контент в читаемой колонке (col-lg-8 col-xl-6)
  - Центрирование через mx-auto
  - Кнопка "Назад" — btn btn-dark
  - Заголовок — display-4 или display-5

### 6. Кастомные стили (опционально)
- [ ] Создать `static/css/style.css`:
  - Переопределить цвета Bootstrap переменных (если нужно)
  - Добавить кастомные классы для акцентов
- [ ] Подключить в base.html

## Результат
Полностью стилизованное приложение с Bootstrap 5, соответствующее style_guide (белый фон, черные кнопки, желтые акценты).

# Фаза 1.4: Создание представлений и шаблонов

## Цель

Реализовать представления списка и детального просмотра постов с шаблонами.

## Задачи

### 1. Базовый шаблон

- [x] Создать `templates/base.html` с:
  - Базовой структурой HTML5
  - Блоками для title, content, scripts
  - Навигацией с Bootstrap 5
  - Подключением статики (Bootstrap 5.3.3, Bootstrap Icons)

### 2. Представление списка постов

- [x] Обновить `post_list()` в `blog/views.py`:
  - Получить все опубликованные посты
  - Передать в контекст шаблона
- [x] Создать `templates/blog/post_list.html`:
  - Наследоваться от base.html
  - Вывести список постов
  - Показать: заголовок, дату, отрывок (первые 50 слов)
  - Ссылка "Читать далее" на детальный просмотр

### 3. Детальное представление поста

- [x] Обновить `post_detail()` в `blog/views.py`:
  - Получить пост по pk
  - Обработать 404 если не найден
- [x] Создать `templates/blog/post_detail.html`:
  - Наследоваться от base.html
  - Полный текст поста
  - Заголовок, дата создания, дата обновления
  - Кнопка "Назад к списку"

### 4. Страница "О блоге"

- [x] Создать простой шаблон `templates/blog/about.html`
- [x] Добавить статический текст о проекте

## Результат

✅ Работающие представления с шаблонами для просмотра списка и детального просмотра постов.
✅ Bootstrap 5 интегрирован через CDN.
✅ Реализован дизайн: белый фон, черные кнопки, Bootstrap Icons.

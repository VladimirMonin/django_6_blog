# Фаза 1.4: Создание представлений и шаблонов

## Цель
Реализовать представления списка и детального просмотра постов с шаблонами.

## Задачи

### 1. Базовый шаблон
- [ ] Создать `templates/base.html` с:
  - Базовой структурой HTML5
  - Блоками для title, content, scripts
  - Заглушкой навигации (пока без стилей)
  - Подключением статики (готовимся к Bootstrap)

### 2. Представление списка постов
- [ ] Обновить `post_list()` в `blog/views.py`:
  - Получить все опубликованные посты
  - Передать в контекст шаблона
- [ ] Создать `templates/blog/post_list.html`:
  - Наследоваться от base.html
  - Вывести список постов
  - Показать: заголовок, дату, отрывок (первые 200 символов)
  - Ссылка "Читать далее" на детальный просмотр

### 3. Детальное представление поста
- [ ] Обновить `post_detail()` в `blog/views.py`:
  - Получить пост по pk
  - Обработать 404 если не найден
- [ ] Создать `templates/blog/post_detail.html`:
  - Наследоваться от base.html
  - Полный текст поста
  - Заголовок, дата создания, дата обновления
  - Кнопка "Назад к списку"

### 4. Страница "О блоге"
- [ ] Создать простой шаблон `templates/blog/about.html`
- [ ] Добавить статический текст о проекте

## Результат
Работающие представления с шаблонами для просмотра списка и детального просмотра постов.
